# Skill: Review Terraform / IaC

## Use when
Reviewing Terraform modules, environment compositions, or pull requests.

## Check for
- module boundaries
- naming consistency
- tagging
- least privilege IAM
- state handling
- dependency clarity
- provider pinning
- destructive replacement risk
- drift risk
- environment promotion suitability

## Output
- Summary of findings
- Critical risks
- Recommended fixes
- Nice-to-have improvements
- Production readiness verdict
