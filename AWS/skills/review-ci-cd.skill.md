# Skill: Review CI/CD Pipeline

## Use when
Reviewing build, test, release, and deployment workflows.

## Check for
- build reproducibility
- test stages and quality gates
- artifact immutability
- secrets handling
- deployment strategy
- rollback path
- environment segregation
- approvals and change controls
- auditability
- observability of pipeline failures

## Deployment patterns to consider
- rolling
- blue/green
- canary
- feature flags
- immutable replacement

## Output
- Pipeline strengths
- Weaknesses / risks
- Missing controls
- Safer rollout recommendations
