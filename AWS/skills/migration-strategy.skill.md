# Skill: Define Migration Strategy

## Use when
Planning migration, modernization, hybrid integration, or workload transition.

## Evaluate
- current state
- target state
- dependency chains
- cutover strategy
- rollback/fallback strategy
- data migration risks
- parallel run requirements
- operational support model

## Output
- migration phases
- risks by phase
- transition architecture
- cutover plan
- rollback options
- readiness criteria
