# Skill: Assess Well-Architected Risks

## Use when
Reviewing an AWS workload for production readiness, architectural weaknesses, or prioritized remediation.

## Evaluate
- operational excellence
- security
- reliability
- performance efficiency
- cost optimization
- sustainability
- workload maturity
- top risk concentration

## Output
- pillar-by-pillar findings
- strengths and weaknesses
- prioritized remediation actions
- temporary risk acceptances
- recommendation for go-live, hardening, or rework
