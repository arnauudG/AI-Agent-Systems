# Playbook: Business Need to Data Product

## Use when
A business problem likely requires a dataset, metric layer, analytics flow, or reusable data product.

## Lead packages
- BUSINESS-ANALYSIS
- DATA

## Suggested sequence
1. frame the problem
2. identify users and decisions improved
3. define MVP scope
4. identify sources and outputs
5. define ownership and quality expectations
6. design the smallest useful data solution
7. record assumptions and trade-offs
8. create data flow diagram if useful
