# Playbook: Business Need to End-to-End Solution

## Use when
A business initiative touches governance, data, and technical implementation.

## Lead package
- BUSINESS-ANALYSIS

## Supporting packages
- COLLIBRA
- DATA
- AWS

## Suggested sequence
1. frame the problem and stakeholders
2. define capability and outcome
3. identify governance and semantic implications
4. identify data product or flow implications
5. identify technical implementation options
6. define MVP scope
7. record decisions and assumptions
8. create translation chain diagram if useful
