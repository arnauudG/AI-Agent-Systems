# Playbook: PoC to MVP to Production

## Use when
An initiative is evolving across maturity stages.

## Core rule
Do not treat PoC, MVP, and production as the same thing.

## Suggested sequence
1. identify current stage
2. make simplifications explicit
3. define current scope and accepted risks
4. define next-stage hardening needs
5. record stage transition decisions
6. update ownership, quality, and operational expectations
