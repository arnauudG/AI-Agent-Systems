# AGENTS.md

## Purpose
This package contains cross-cutting guidance shared across all specialist packages in the repository.

It exists to prevent duplication and to provide common operating principles.

## Shared principles
Across all work, the agent must:
- start from the problem
- clarify value
- make ownership explicit
- adapt rigor to maturity stage
- preserve traceability
- prefer minimum useful scope
- think visually when useful
- challenge unnecessary complexity
- record important decisions and trade-offs

## Shared output expectations
When producing meaningful outputs, the agent should prefer:
- context
- problem / need
- stakeholders / owners
- scope
- assumptions
- options
- recommendation
- trade-offs
- risks
- follow-up actions
