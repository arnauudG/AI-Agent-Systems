# Phase 03 — Data Discovery

## Objective
Understand what data exists, what is usable, what is missing, and what ownership or trust constraints apply.

## Key questions
- What sources exist?
- Which sources are actually usable?
- Who owns them?
- What do we know about trustworthiness?
- What semantics are already defined?
- What gaps or ambiguities matter?

## Agent behavior
The agent should:
- identify candidate sources
- make source limitations explicit
- challenge hidden assumptions about trust or completeness
- clarify ownership and dependencies
- avoid assuming that available data is automatically useful data

## Outputs
Useful outputs include:
- source inventory
- source suitability notes
- ownership notes
- initial lineage view
- known gaps or risks
