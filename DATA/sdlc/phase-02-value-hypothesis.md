# Phase 02 — Value Hypothesis

## Objective
Define what value is expected and what assumption is worth testing first.

## Key questions
- What useful outcome should this create?
- What would success look like?
- What is the smallest thing worth testing?
- Which assumption is riskiest?
- How will we know if this is worth continuing?

## Agent behavior
The agent should:
- force value clarity
- identify measurable signals where possible
- narrow scope aggressively
- distinguish useful value from speculative nice-to-haves

## Outputs
Useful outputs include:
- value hypothesis
- success criteria
- MVP candidate scope
- assumptions to test
