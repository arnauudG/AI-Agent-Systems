# SVG Layout and Spacing Rules

## Goal
Make diagrams visually balanced and easy to scan.

## Rules
- Use grid-like alignment whenever possible.
- Preserve generous whitespace between sections.
- Keep consistent vertical rhythm.
- Align related blocks to a common baseline or edge.
- Avoid near-overlaps and cramped labels.
- Prefer wider layouts over compressed layouts when readability matters.
- Keep connector lines simple and intentional.

## Heuristics
- Titles should have clear separation from body content.
- Column layouts should maintain consistent gutters.
- Text blocks should not touch visual separators.
- Use repeated spacing values where possible.
