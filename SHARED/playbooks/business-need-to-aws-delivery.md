# Playbook: Business Need to AWS Delivery

## Use when
A business need is already fairly clear and the main question is how to implement it on AWS.

## Lead package
- AWS

## Supporting package
- BUSINESS-ANALYSIS if scope or stakeholders still need clarification

## Suggested sequence
1. confirm business outcome
2. identify constraints and NFRs
3. define technical implications
4. generate architecture options
5. choose delivery and operational model
6. record decisions
7. create architecture diagram if useful
