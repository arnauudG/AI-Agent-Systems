# Ownership Model Checklist

Use this checklist to review whether an ownership model is realistic and useful.

## Role clarity
- [ ] Roles are clearly defined.
- [ ] Role names reflect real responsibilities.
- [ ] Different responsibility types are not collapsed carelessly.

## Accountability
- [ ] Each major responsibility has an accountable role.
- [ ] Assigned roles have actual authority or influence.
- [ ] Ownership is not decorative.

## Operational realism
- [ ] The required actors actually exist.
- [ ] The responsibilities are feasible in practice.
- [ ] Escalation or ambiguity handling is clear.

## Collibra fit
- [ ] The ownership model maps cleanly enough into Collibra.
- [ ] Assignments support real governance use.
- [ ] The model is not more complex than necessary.

## Maintainability
- [ ] The model can be explained to users and maintainers.
- [ ] The model is durable enough for the maturity stage.
- [ ] Temporary ownership gaps are explicitly documented.

## Final question
- [ ] Does this reflect how responsibility actually works, or only how we wish it worked in a PowerPoint kingdom?
