# MVP Scope Checklist

Use this checklist to review whether the proposed MVP scope is truly minimal, valuable, and credible.

## Core value
- [ ] The MVP objective is explicit.
- [ ] The first audience is explicit.
- [ ] The scope supports a real outcome rather than symbolic progress.

## Minimality
- [ ] Must-have scope is clearly separated from later-phase scope.
- [ ] Nice-to-have items were challenged.
- [ ] Multi-audience scope inflation was challenged.
- [ ] The MVP does not attempt to solve every adjacent problem.

## Realism
- [ ] The scope looks deliverable within realistic constraints.
- [ ] Dependencies are visible enough.
- [ ] Accepted simplifications are explicit.
- [ ] The support and ownership model is believable for the stage.

## Learning value
- [ ] The MVP will test meaningful assumptions.
- [ ] Success signals are visible.
- [ ] Failure would still teach something useful.

## Final question
- [ ] Is this truly an MVP, or a politically negotiated medium-sized release wearing a tiny hat?
