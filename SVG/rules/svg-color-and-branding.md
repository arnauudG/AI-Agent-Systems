# SVG Color and Branding Rules

## Goal
Use color consistently and purposefully without harming legibility or semantics.

## Rules
- Use a limited, consistent palette.
- Reserve color for hierarchy, emphasis, or semantic meaning (e.g. trust boundaries, flow direction).
- Ensure sufficient contrast for text and key elements.
- Prefer semantic color use over purely decorative use.
- When a brand palette exists, use it for key elements; keep diagrams readable when printed in grayscale if needed.

## Heuristics
- Same concept (e.g. “owned by team A”) should use the same color across the diagram.
- Avoid many similar shades that are hard to distinguish.
- Use color in legends when color carries meaning.
